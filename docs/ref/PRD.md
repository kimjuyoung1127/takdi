# Takdi PRD

Last Updated: 2026-03-05 (KST)

## Product Goal
- Generate high-quality product detail pages from brief text and image assets.
- Prioritize one-person daily usability before subscription expansion.

## Primary User
- Single operator who repeatedly produces e-commerce detail content.

## Core Workflow
1. Upload text brief and images.
2. Run AI analysis and draft generation.
3. Review and edit generated sections.
4. Export detail-page outputs.

## MVP Scope
- Included:
  - Project CRUD
  - Text and image input
  - AI draft generation
  - Manual edit
  - Export
  - Usage summary
- Excluded:
  - Team invite and role UI
  - Live billing integration
  - Template marketplace

## Validation Gate
- 20+ real outputs completed.
- Reuse intent confirmed.
