- 모드 카드, 헤더, 사이드바, `/projects`, `/settings` 문구 한국어화

## 자기리뷰
- 홈과 `/projects`, `/settings`는 최신 목록이 반영되도록 기존 `force-dynamic` 설정 유지
- 프로젝트와 템플릿은 같은 리스트로 섞지 않고 별도 섹션 유지
- 탭과 필터 값은 내부 enum을 유지하고 표시 라벨만 메시지 사전으로 분리
