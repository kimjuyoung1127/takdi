# 탁디장 스튜디오 — 야간 자동화 실행 로그

> 최종 갱신일: 2026-03-05

---

## 현재 상태

아직 실행 기록 없음. 자동화 프로세스가 설정되면 이 문서에 실행 로그가 자동 기록된다.

---

## 로그 형식 정의

### 개별 실행 로그

```
### {YYYY-MM-DD} {HH:mm}

| 프로세스 | 상태 | 소요시간 | 작업 내용 |
|----------|------|----------|-----------|
| {프로세스명} | {성공/실패/건너뜀} | {N.Ns} | {수행한 작업 요약} |
```

**상태 값:**
- `성공` — 정상 완료
- `실패` — 오류 발생 (오류 내용을 작업 내용에 기재)
- `건너뜀` — 조건 미충족으로 실행하지 않음

### 상세 로그 (오류 시)

```
#### {프로세스명} 오류 상세

- 오류 유형: {유형}
- 오류 메시지: {메시지}
- 영향 범위: {영향받는 문서/파일 목록}
- 조치 필요: {예/아니오}
- 조치 내용: {필요 시 기재}
```

## 주간 요약 형식

매주 월요일 자동 생성되는 주간 요약 테이블:

```
## 주간 요약: {YYYY}-W{WW}

| 프로세스 | 실행 횟수 | 성공 | 실패 | 건너뜀 | 평균 소요시간 |
|----------|-----------|------|------|--------|---------------|
| docs-nightly-organizer | {N} | {N} | {N} | {N} | {N.Ns} |
| code-doc-align | {N} | {N} | {N} | {N} | {N.Ns} |
| integrity-report | {N} | {N} | {N} | {N} | {N.Ns} |

주요 이슈:
- {이슈 요약 또는 "없음"}
```
