# 🚀 [내부 업무용 영상 자동화 시스템] 상세 스펙 명세서

## 1. 시스템 개요 (Overview)

본 시스템은 디자이너가 보유한 기존 하드웨어(Mac Mini + NAS)를 100% 활용하여, 기획 텍스트와 원본 이미지 몇 장만으로 **다매체 숏폼 영상(3종)과 썸네일, 마케팅 스크립트를 원클릭으로 자동 생성**하는 완전 독립형 사내 렌더링 엔진입니다. 외부 클라우드나 유료 API를 전혀 사용하지 않아 **운영 유지비가 0원**인 것이 핵심입니다.

## 2. 하드웨어 및 인프라 스펙

- **메인 렌더링 서버:** Apple Mac Mini (로컬 네트워크 구동)
- **메인 스토리지:** Synology NAS (원격 마운트 및 자동 저장소)
- **운영 환경:** 폐쇄형 로컬 네트워크(또는 Tailscale을 이용한 외부 보안 접속)

## 3. 핵심 기능 스펙 (Core Features)

### ① 직관적인 웹 대시보드 (입력부)

- **기능:** 복잡한 프리미어/애프터이펙트 화면 대신, 심플한 웹페이지(Next.js)에서 모든 작업을 통제합니다.
- **입력 항목:**
    - **텍스트:** 제품명, 메인 타깃, 문제/해결 베네핏, 핵심 소구점(USP), 행동 촉구(CTA)
    - **이미지:** 메인 컷, 문제 상황 컷, 해결 컷 등 폴더 규격에 맞춘 이미지 5~6장 (드래그 앤 드롭 지원)
    - **옵션 선택:** 템플릿 무드 선택, BGM 선택

### ② AI 자동 비율 변환 (Smart Cropping)

- **기능:** 가로/세로 원본 이미지의 비율에 얽매이지 않고, AI가 피사체(상품, 인물 얼굴)의 중심을 찾아냅니다.
- **효과:** 디자이너가 9:16, 1:1, 16:9 비율별로 이미지를 일일이 자르고 위치를 맞추는 수작업(노가다)을 로컬 비전 AI(OpenCV/YOLO)가 1초 만에 자동 처리합니다.

### ③ 텍스트-비트 동기화 렌더링 엔진 (Remotion)

- **기능:** 코드로 영상을 합성하는 엔진이 맥 미니 백그라운드에서 구동됩니다.
- **타이밍 자동화 (TTS 대체 로직):**
    - **가독성 계산:** 입력된 자막의 글자 수를 계산하여 화면이 머무는 시간(Duration)을 자동으로 늘리거나 줄입니다.
    - **비트 스냅(Beat Snap):** NAS에 저장된 BGM의 하이라이트/비트를 시스템이 미리 분석하여, 화면 전환(Transition)이 음악의 박자에 딱 맞게 떨어지도록 강제 보정합니다.

### ④ 바이럴 스크립트 자동 생성 (Local LLM)

- **기능:** 맥 미니 내부에서 무료 AI 언어 모델(Ollama)이 구동됩니다.
- **효과:** 영상에 쓰인 기획안 텍스트를 분석하여, 인스타그램 릴스나 틱톡 본문에 복사해서 붙여넣기만 하면 되는 '후킹 멘트 + 본문 + 해시태그' 세트를 즉각 생성합니다.

## 4. 자동화 산출물 스펙 (Output)

버튼 한 번 클릭 후 1~2분 내에 아래의 결과물이 **NAS의 [프로젝트명_날짜] 폴더**에 자동 정리됩니다.

1. **숏폼 영상 (9:16 비율):** 릴스, 쇼츠, 틱톡용 15초 세로형 MP4
2. **피드 영상 (1:1 비율):** 인스타그램 일반 피드용 15초 정사각형 MP4
3. **롱폼 영상 (16:9 비율):** 유튜브/웹사이트 상세페이지 삽입용 가로형 MP4
4. **마케팅 썸네일 (JPG):** 메인 컷을 바탕으로 후킹 텍스트가 합성된 고화질 썸네일 이미지
5. **업로드 스크립트 (TXT):** 해시태그가 포함된 소셜 미디어 배포용 텍스트 문서

## 5. 기대 효과 및 ROI (Return on Investment)

- **작업 시간 90% 단축:** 1건당 최소 1~2시간 소요되던 컷편집, 비율 변환, 타이밍 동기화, 랜더링 대기 시간이 **'웹 폼 입력 후 2분 대기'**로 압축됩니다.
- **무한 렌더링 (비용 Zero):** 외부 비디오 API(Creatomate 등) 사용 시 발생하는 건당 과금이나, 챗GPT 월 구독료가 전혀 발생하지 않습니다. 맥 미니를 혹사시켜도 추가 비용은 전기세뿐입니다.
- **완벽한 에셋 자산화:** 디자이너의 PC 바탕화면이 지저분해질 일이 없습니다. 모든 입력 소스와 최종 결과물은 NAS에 체계적으로 자동 분류 및 아카이빙됩니다.
